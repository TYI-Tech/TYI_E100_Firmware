# 使用说明

以下命令是客户日常使用的主要运维入口。

## 启动或重新部署

```bash
bash ./scripts/deploy.sh
```

## 仅重新构建

```bash
bash ./scripts/build.sh
```

## 启动或停止容器

```bash
bash ./scripts/up.sh
bash ./scripts/down.sh
```

## 查看状态

```bash
bash ./scripts/status.sh
bash ./scripts/logs.sh
bash ./scripts/enter.sh
```

`enter.sh` 会进入 `base-stack` 容器内部。
容器内默认运行用户为 `uav`，便于日常运维时避免使用 root。
