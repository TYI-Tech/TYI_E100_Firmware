# 常见问题

## 首先应该修改哪个文件？

`machine.env`

该文件是主要的可编辑入口，用于设置 UART、MID360 序列号/IP 和宿主机网卡参数。

## 是否需要修改 `.env`？

通常不需要。

`.env` 主要保存部署默认值，并会由 `scripts/configure_machine.sh` 自动更新。

## 默认是拉镜像还是本地构建？

默认拉取预编译镜像。

执行 `bash ./scripts/deploy.sh` 时，会直接使用 ACR 中的预编译镜像。
如需切换为本地源码构建，可执行 `bash ./scripts/deploy.sh --build`。

## 使用预编译镜像前是否需要登录镜像仓库？

通常需要。

可先执行：

```bash
docker login --username=hyzrichard crpi-zpvbhgsm3t97idht.cn-hangzhou.personal.cr.aliyuncs.com
```

如果现场环境不方便访问镜像仓库，可直接改用 `bash ./scripts/deploy.sh --build`。

## 是否还依赖在线下载 GeographicLib 数据？

默认不依赖。

当前版本已将构建所需的 `egm96-5` geoid 文件随仓库提供，避免构建过程中额外访问 SourceForge 下载该资源。

## 构建时是否仍需要联网？

需要。

构建仍需要访问 Ubuntu/ROS 软件源安装系统依赖，但基础安装流程已经加入自动重试，以降低瞬时 DNS 或镜像波动的影响。

## LIO 输入话题或参考坐标系能否修改？

可以。

如后续接入控制桥接，可通过 `configs/fastlio_to_mavros/bridge.yaml` 调整桥接输入话题和参考坐标系。

## `TYI_Plugin_Ctl` 是否包含在本仓库中？

不包含。

本仓库仅提供 firmware 侧运行链路。
`TYI_Plugin_Ctl` 通过独立的 apt 包分发。
