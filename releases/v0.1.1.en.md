# TYI E100 Firmware v0.1.1

## Release Summary

`v0.1.1` closes the clean source-build workflow for the product firmware repository and completes the first release-oriented documentation pass.
This version has been validated on the `uav-nx` board for both image build and runtime bring-up.

## Main Updates

- supports `docker compose build --no-cache base-stack` directly from the checked-out source tree
- fixes ROS1 manifest discovery for `livox_ros_driver2` during fresh builds
- bundles the required GeographicLib `egm96-5` geoid dataset with the repository
- adds retry handling for base `apt` dependency installation during image build
- completes the homepage, bilingual documentation, release notes, release process, and release check tooling

## Validated Scope

- runtime chain:
  `livox_lidar_publisher2 -> robot/dlio_odom -> fastlio_to_mavros -> mavros`
- key topics:
  `/robot/dlio/odom_node/odom`
  `/mavros/vision_pose/pose`
- production runtime container:
  `uav-base-stack-customer`

## Usage Guidance

- start with `README.md` and `docs/en_US/quick_start.md`
- keep machine-specific differences inside `machine.env`
- continue to install `TYI_Plugin_Ctl` separately when control bridge capability is required
