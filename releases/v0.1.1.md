# TYI E100 Firmware v0.1.1

## 发布概览

`v0.1.1` 主要完成了产品固件仓库的源码构建闭环与发布整理。
当前版本已经在 `uav-nx` 板卡上完成实机构建与运行验证。

## 主要更新

- 支持从仓库源码直接执行 `docker compose build --no-cache base-stack`
- 修复 fresh build 场景下 `livox_ros_driver2` 的 ROS1 清单识别问题
- 将 GeographicLib 所需 `egm96-5` geoid 数据随仓库提供，避免额外在线下载该资源
- 为镜像构建阶段的基础 `apt` 依赖安装增加自动重试
- 完成首页、双语文档、版本说明、发布流程与发布检查脚本整理

## 已验证内容

- 运行链路：
  `livox_lidar_publisher2 -> robot/dlio_odom -> fastlio_to_mavros -> mavros`
- 关键话题：
  `/robot/dlio/odom_node/odom`
  `/mavros/vision_pose/pose`
- 正式运行容器：
  `uav-base-stack-customer`

## 使用建议

- 首次部署优先阅读 `README.md` 与 `docs/zh_CN/快速上手.md`
- 机型差异建议只通过 `machine.env` 管理
- 如需控制桥接能力，继续通过独立的 `TYI_Plugin_Ctl` 包提供
